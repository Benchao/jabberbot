# Plymouth State University Jabber Bot

## npm Requirements
### [icu4c](http://userguide.icu-project.org/)
On a mac, install with brew!
````
brew install icu4c
brew link icu4c
````

### simple-xmpp
Install via: `npm install -g simple-xmpp`

## Running the Bot

Run this bad boy like so:

````
node bot.js
````
