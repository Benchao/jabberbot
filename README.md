# Plymouth State University Jabber Bot

## Requirements
### node (of course)
On a mac, install with brew!

````
brew install node
````

### npm (of course)
Run:
````
curl http://npmjs.org/install.sh | sh
````

### [icu4c](http://userguide.icu-project.org/)
On a mac, install with brew!
````
brew install icu4c
brew link icu4c
````

### npm: simple-xmpp
Install via: `npm install -g simple-xmpp`

## Running the Bot

Run this bad boy like so:

````
node bot.js
````
